# OSRS Discord Bot
**A Discord bot with some OSRS utilities.**

A **secrets.json** file should be created in the root folder with the following content filled properly.
```JSON
{
    "BOT_TOKEN": "",
    "WOM_API_KEY": "",
    "WOM_GROUP": 1234,
    "UPDATE_HS_EVERY_X_MIN": 10,
    "HS_CHANNEL_ID": 123456789
}
```
